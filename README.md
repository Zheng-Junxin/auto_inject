# 简历精准投递 Edge 插件

这是一个可直接加载的 Microsoft Edge / Chromium Manifest V3 扩展，用于在招聘网站页面内做简历解析、本地匹配、LLM 匹配、表单填充和受限自动投递。

## 支持站点

- BOSS 直聘：`zhipin.com`
- 前程无忧 51Job：`51job.com`
- 智联招聘：`zhaopin.com`
- 猎聘：`liepin.com`
- 脉脉：`maimai.cn`
- 神仙外企：`waiqi.com`、`offersir.com`

## 安装

1. 打开 Edge：`edge://extensions/`
2. 开启“开发人员模式”
3. 选择“加载解压缩的扩展”
4. 选择本仓库的 `extension` 目录

## 功能

- 上传解析简历：支持 `txt`、`md`、`json`、`docx`、`pdf`，自动提取姓名、手机、邮箱、求职方向、城市和技能。
- 本地匹配：根据关键词、排除词、城市和简历摘要生成本地评分。
- LLM 匹配：支持 OpenAI-compatible `base_url`、`model`、`API key`，由后台服务调用，不把 key 暴露给内容脚本。
- 页面填充：在职位详情或申请表单中填充联系方式、城市、薪资和沟通模板。
- 自动采集岗位：在当前搜索结果页自动滚动、点击“加载更多”，并在可识别下一页链接时继续采集。
- 自动处理本页：先自动采集当前搜索结果，再按简历、本地规则和 LLM 分数筛选，后台只复用一个工作标签页逐个进入岗位详情并尝试填充和投递，避免批量打开重复标签页。

## 使用流程

1. 点击扩展图标，进入“设置资料”。
2. 上传简历并点击“解析并填充”，检查自动提取的信息。
3. 配置关键词、排除词、城市和每日投递上限。
4. 如需 LLM 匹配，填写 `base_url`、`model`、`API key`，勾选发送授权，再点击“授权并测试 LLM”。
5. 打开招聘网站搜索结果页，点击“自动采集岗位”；插件会自动滚动、加载更多并尝试翻页。
6. 需要复核时点击“LLM 重评”；需要直接执行时点击“匹配并自动投递”。这一次点击会作为本轮明确授权，插件会基于已上传/填写的简历信息匹配岗位并投递。
7. 如需无人值守批量队列，可额外启用“自动处理队列”和“允许自动点击投递/沟通按钮”，仍受每日上限、去重和阻断停止限制。

## 自动采集参数

设置页的“自动化”区域可控制采集强度：

- `采集滚动次数`：每个结果页最多向下滚动多少次。
- `采集页数`：最多采集多少个结果页。
- `滚动等待毫秒`：每次滚动或加载更多后的等待时间。
- `采集时尝试进入下一页`：识别到下一页链接时自动跳转继续采集。
- `自动投递前先滚动采集当前搜索结果页`：点击“自动处理本页”时先执行采集，再生成投递队列。

## 安全边界

- 简历、联系方式、API key 和投递记录保存在 `chrome.storage.local`。
- LLM 请求只发往你配置并授权的 `base_url`。
- 非本机 `http` 地址会被拒绝，避免明文传输 API key。
- 不绕过验证码、登录、滑块、安全验证、付费墙或招聘网站风控。
- 自动化遇到登录、验证码、频控或资料要求时会停止。
- 自动投递可能违反部分网站规则，请只在你确认允许的范围内使用。
